## Penjelasan singkat tentang jaga mental
Jaga Mental adalah aplikasi web kesehatan mental yang membantu pengguna menyadari, mengelola, dan meningkatkan kondisi mental mereka melalui fitur mood tracking, self-care, meditasi, serta dukungan profesional. Aplikasi ini menekankan kesadaran diri (awareness), pengelolaan mandiri (management), dan dukungan profesional (support) dengan pendekatan yang aman, personal, dan berbasis data.

# Aplikasi ini dibuat karena:
1. Banyak orang mengalami masalah kesehatan mental tapi sulit mengakses bantuan.
2. Stigma dan keterbatasan layanan profesional masih tinggi.
3. Generasi muda lebih nyaman dengan solusi digital.

# Tujuan utama Jaga Mental:
1. Meningkatkan kesadaran kondisi emosional pengguna
2. Membantu mengelola stres dan emosi
3. Memberikan data pendukung untuk intervensi dini

## Tiga Pilar Utama
# Pilar 1: Awareness (Kesadaran Diri)

1. Fokus pada mengenali emosi:
2. Mood tracker harian
3. Analisis tren dan pola emosi
4. Membantu pengguna memahami trigger stres atau kebahagiaan


# Pilar 2: Management (Pengelolaan Diri)

1. Fokus pada kebiasaan sehat:
2. Self-care tracker (makan, tidur, olahraga, sosial, hobi)
3. Meditasi terpandu
4. Wellness score & rekomendasi personal

# Pilar 3: Support (Dukungan Profesional)

1. Fokus pada bantuan lanjutan:
2. Konsultasi dengan profesional
3. Laporan kesehatan mental berbasis data
4. Artikel edukatif & resource hub
